Web3 - Multi-hostname Website

This is a test website for caddy-gen. It's configured with multiple hostnames and directives.

Configuration: virtual.bind: 80 web3.local www.web3.local | host:tls internal | header Strict-Transport-Security "max-age=31536000"

Hostnames: web3.local, www.web3.local

Host Directive: tls internal

Proxy Directive: header Strict-Transport-Security "max-age=31536000"
